# M-04 — Run all N×(N-1)/2 pairs

## Goal
Every file pair is compared, not just adjacent ones. Non-consecutive overlaps (A overlaps C) are detected.

## Steps
1. Generate all pairs: `itertools.combinations(video_files, 2)`
2. For each pair: run M-01 → M-02 → M-03
3. Collect all non-None `OverlapMatch` results into an `OverlapGraph` (just a list for MVP)
4. Log each detected overlap with file names, range, and confidence

## Notes
- O(N²) pairs is fine — N is number of input files, not frames.
- Pairs can run concurrently: `asyncio.gather(*[match_pair(a, b) for a, b in pairs])`
- Known file order is used in T-01/T-02, not here. Match phase is order-agnostic.

## Done when
Given 3 files where A overlaps C (not B), the `OverlapMatch` for (A, C) is found and (A, B) / (B, C) return None.
