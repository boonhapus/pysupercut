# PO-05 — README + install instructions

## Goal
Someone new can clone the repo and run the tool without asking questions.

## Sections

### Requirements
- Python 3.14+
- uv
- ffmpeg + ffprobe on $PATH
- fpcalc (chromaprint) on $PATH

### Install
```bash
# Install fpcalc
brew install chromaprint          # macOS
sudo apt install libchromaprint-tools  # Ubuntu

# Install ffmpeg
brew install ffmpeg               # macOS
sudo apt install ffmpeg           # Ubuntu

# Install tool
git clone <repo>
cd pysupercut
uv sync
```

### Usage
```bash
# Deduplicate and stitch
uv run pysupercut a.mp4 b.mp4 c.mp4 --output result.mp4

# Preview plan without running ffmpeg
uv run pysupercut a.mp4 b.mp4 c.mp4 --dry-run

# Tune sensitivity
uv run pysupercut *.mp4 --min-overlap 10 --phash-threshold 8 --audio-threshold 0.90
```

### How it works
Brief 3–4 sentence explanation of the pipeline: probe → fingerprint → match → timeline → stitch.

### Limitations
- All input files must share the same video codec (e.g. all H.264)
- Files must be passed in chronological order
- Requires both audio and video streams for highest confidence matching

## Done when
A teammate can follow the README cold and successfully run the tool on test files.
