# ST-04 — Verify output duration + cleanup

## Goal
Confirm output is correct, then clean up all temp files.

## Steps
1. ffprobe output file → parse duration
2. Compare to `total_output` from T-05
3. If `abs(actual - expected) > 0.5` → log warning with both values (don't error — encoding variance is real)
4. Log final summary:
   ```
   Output: output.mp4  (205.1s, 847 MB)
   Removed: 110.0s of duplicate content across 3 overlaps
   ```
5. Call `tempdir.cleanup()` → removes all temp re-encoded segments + concat list

## Notes
- Cleanup must happen even if verification fails — use try/finally.
- If user wants to inspect temp files for debugging, they can use `--dry-run` instead.

## Done when
Temp directory is gone after run. Output file duration logged. Warning fires if duration drifts > 0.5s.
