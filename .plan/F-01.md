# F-01 — Audio fingerprinting via fpcalc

## Goal
Each file produces a list of `AudioChunk` objects with start time + raw chromaprint fingerprint.

## Data structure
```python
@define
class AudioChunk:
    start: float
    fingerprint: list[int]   # raw chromaprint int array
```

## Steps
1. Run: `fpcalc -raw -chunk 5 -overlap <file>` via `asyncio.create_subprocess_exec`
2. Parse stdout: each line is `FINGERPRINT=<comma-separated ints>` preceded by `DURATION=` and offset metadata
3. Map to `AudioChunk` list
4. Skip (log warning) if `VideoFile.has_audio == False`

## Notes
- Chunk size 5s aligns with default `--min-overlap`. If user sets `--min-overlap` lower, chunk size should match.
- `fpcalc -overlap` produces overlapping windows — useful for catching overlaps that straddle chunk boundaries.

## Done when
Given a video file, function returns a non-empty list of `AudioChunk` with correct `start` values.
