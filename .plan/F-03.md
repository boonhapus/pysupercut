# F-03 — Run fingerprinting concurrently per file

## Goal
Audio + video fingerprinting runs concurrently per file, and all files fingerprinted concurrently.

## Steps
1. Create `fingerprint_file(vf: VideoFile) -> tuple[list[AudioChunk], list[FrameHash]]`
2. Inside: `await asyncio.gather(fingerprint_audio(vf), fingerprint_video(vf))`
3. Top-level: `await asyncio.gather(*[fingerprint_file(vf) for vf in video_files])`
4. Return dict `{path: (chunks, hashes)}`

## Notes
- ffmpeg/fpcalc subprocesses are I/O-bound — `asyncio.gather` is appropriate.
- Hash compute loop inside F-02 uses `asyncio.to_thread` — that's already handled there.

## Done when
N files fingerprinted in roughly the time of the slowest single file (not N × single file time).
Confirm via `time` on 3 files vs 1 file.
