# T-04 — Emit ordered Segment list

## Goal
Produce the final ordered list of `Segment` objects ready for stitching.

## Data structure
```python
@define
class Segment:
    source: Path
    start: float
    end: float
    needs_reencode: bool    # True if start > 0 or end < duration
```

## Steps
1. Iterate files in known (user-supplied) order
2. Skip DROPPED files
3. For each remaining file: emit `Segment(source=path, start=keep_start, end=keep_end)`
4. Set `needs_reencode = (start > 0.0 or end < duration)`

## Done when
Given files [A, B, C] where B is trimmed at head by 30s:
- Segment A: start=0, end=A.duration, needs_reencode=False
- Segment B: start=30, end=B.duration, needs_reencode=True
- Segment C: start=0, end=C.duration, needs_reencode=False
