# ST-02 — Build concat demuxer list

## Goal
Write an ffmpeg concat demuxer input file listing all segments in order.

## Format
```
# ffmpeg concat demuxer list
file '/abs/path/to/a.mp4'
file '/tmp/tmpXXXX/c_40_70.mp4'
file '/tmp/tmpXXXX/d_40_95.mp4'
```

## Steps
1. Iterate `Segment` list in order
2. For `needs_reencode=False` → use original `segment.source` (absolute path)
3. For `needs_reencode=True` → use temp path from ST-01 map
4. Write to `<tempdir>/concat_list.txt`
5. All paths must be absolute — use `Path.resolve()`

## Notes
- `-safe 0` flag in ST-03 allows absolute paths in the list file.
- Paths with spaces must be quoted — Python's f-string with single quotes handles this as long as filenames don't contain single quotes. Log a warning if any path contains `'`.

## Done when
`concat_list.txt` written with correct absolute paths. Each path verified to exist before writing.
