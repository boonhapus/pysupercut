# F-02 — Video frame hashing via dhash

## Goal
Each file produces a list of `FrameHash` objects at the configured sample rate.

## Data structure
```python
@define
class FrameHash:
    timestamp: float
    value: int    # 64-bit dhash as int
```

## Steps
1. Run ffmpeg to pipe rawvideo at `--fps` (default 1fps):
   ```
   ffmpeg -i <file> -vf fps=<fps> -f rawvideo -pix_fmt rgb24 pipe:1
   ```
   via `asyncio.create_subprocess_exec` with `stdout=PIPE`
2. Read stdout in chunks of `width * height * 3` bytes per frame
3. Need frame dimensions from `VideoFile` (add `width: int`, `height: int` to VideoFile in P-01 if not already)
4. Convert bytes → PIL Image → `imagehash.dhash(img)` → store `.hash.flatten()` as int via `int(str(h), 16)` or numpy packbits
5. Timestamp = `frame_index / fps`

## Notes
- 1fps on a 1hr file = 3600 frames. Memory is fine; this is ~3600 × 8 bytes for the hashes.
- Use `asyncio.to_thread` for the hash compute loop — it's CPU-bound.

## Done when
Given a video file, returns non-empty list of `FrameHash` with correct timestamps. Spot-check: frame count ≈ `duration * fps`.
