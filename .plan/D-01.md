# D-01 — --dry-run output

## Goal
`--dry-run` prints the full timeline plan and exits without touching ffmpeg.

## Output format
```
=== video-dedup dry run ===

Input files (4):
  [0] a.mp4        duration=120.0s
  [1] b.mp4        duration=90.0s   DROPPED (fully contained in a.mp4)
  [2] c.mp4        duration=110.0s
  [3] d.mp4        duration=95.0s

Overlaps detected (2):
  a.mp4 [80.0–120.0s] ↔ c.mp4 [0.0–40.0s]   confidence=0.94
  c.mp4 [70.0–110.0s] ↔ d.mp4 [0.0–40.0s]   confidence=0.91

Segments to stitch (3):
  1. a.mp4        0.0s → 120.0s   (full, copy)
  2. c.mp4        40.0s → 70.0s   (trimmed, re-encode)
  3. d.mp4        40.0s → 95.0s   (trimmed, re-encode)

Expected output duration: 205.0s
Total time saved: 110.0s (48% deduplication)
```

## Steps
1. Print input files + DROPPED annotations
2. Print all `OverlapMatch` edges with confidence
3. Print `Segment` list with copy vs re-encode label
4. Print expected output duration + savings
5. Exit 0 — no ffmpeg invoked

## Done when
`--dry-run` on a known test set prints correct segments and duration. Verify savings = total_input - total_output.
