# P-01 — ffprobe files async → VideoFile

## Goal
Each input file is probed and represented as a validated `VideoFile` attrs object.

## Data structure
```python
@define
class VideoFile:
    path: Path
    duration: float
    has_audio: bool
    codec: str        # e.g. "h264"
    container: str    # e.g. "mp4"
```

## Steps
1. Build async fn `probe_file(path: Path) -> VideoFile`
2. Run: `ffprobe -v quiet -print_format json -show_streams -show_format <file>`
3. Parse JSON → populate VideoFile
4. Hard error if file not found or ffprobe returns non-zero
5. Run all probes concurrently: `await asyncio.gather(*[probe_file(f) for f in files])`

## Done when
Given 2–3 test video files, all `VideoFile` objects print with correct duration, codec, has_audio.
