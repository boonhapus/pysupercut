# S-01 — uv init + add all deps

## Goal
Project scaffolded, all deps installable, imports resolve.

## Steps
1. `uv init video-dedup`
2. `uv add structlog attrs cattrs cyclopts pyacoustid imagehash numpy`
3. Confirm `uv run python -c "import structlog, attrs, cattrs, cyclopts, acoustid, imagehash, numpy"` exits 0

## Notes
- `pyacoustid` is the Python wrapper for chromaprint/fpcalc — it does NOT ship the `fpcalc` binary itself. That's handled in S-02.
- `imagehash` provides dhash for video frame comparison.

## Done when
`uv run python -c "..."` imports all deps with no errors.
