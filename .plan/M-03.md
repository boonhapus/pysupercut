# M-03 — Combine audio + video ranges → OverlapMatch

## Goal
Merge `AudioMatch` and `VideoMatch` for a pair into a single `OverlapMatch`, or discard if signals disagree.

## Data structure
```python
@define
class OverlapMatch:
    file_a: Path
    file_b: Path
    range_a: tuple[float, float]
    range_b: tuple[float, float]
    confidence: float    # 1 - (norm_audio_dist*0.5 + norm_video_dist*0.5)
```

## Logic
1. If both `AudioMatch` and `VideoMatch` found:
   - Check ranges agree within `--min-overlap` seconds
   - If yes → compute confidence, emit `OverlapMatch`
   - If no → log disagreement + skip (conservative)
2. If only one signal found → log warning, skip (require both for MVP)
3. If neither → no overlap for this pair

## Notes
- Confidence formula: `1 - ((1 - audio_score) * 0.5 + (1 - video_score) * 0.5)`
- "Agree within min-overlap" = `abs(audio_midpoint - video_midpoint) < min_overlap`

## Done when
Given a matching pair, emits `OverlapMatch` with confidence > 0.
Given a non-matching pair, returns `None`.
