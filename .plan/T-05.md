# T-05 — Verify segment duration invariant

## Goal
Sanity check before stitch: total segment duration is less than total input duration when overlaps were found.

## Steps
1. `total_input = sum(vf.duration for vf in video_files)`
2. `total_output = sum(seg.end - seg.start for seg in segments)`
3. If any overlaps found: assert `total_output < total_input` → hard error if not
4. Log both values regardless

## Notes
- If zero overlaps found: `total_output == total_input` is correct — don't error.
- Tolerance: allow `total_output <= total_input + 0.5` (float imprecision).

## Done when
With 30s overlap between 2 files of 60s each → total_output = 90s < total_input = 120s. ✓
With no overlaps → total_output ≈ total_input. ✓
