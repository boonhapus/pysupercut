# S-03 — Stub CLI with cyclopts

## Goal
CLI entry point accepts all flags and files, prints parsed args, exits cleanly.

## Steps
1. Create `cli.py` with cyclopts `App`
2. Define command accepting `files: list[Path]` + all flags
3. Print parsed values and return — no logic yet

## CLI surface
```
dedup-videos [FILES]...
  --min-overlap FLOAT   default: 5.0    (seconds)
  --phash-threshold INT default: 10     (hamming distance 0–64)
  --audio-threshold FLT default: 0.85
  --fps FLOAT           default: 1.0    (frame sample rate for hashing)
  --output PATH         default: output.mp4
  --dry-run             flag, default False
```

## Done when
`uv run dedup-videos a.mp4 b.mp4 --min-overlap 10 --dry-run` prints all parsed values and exits 0.
