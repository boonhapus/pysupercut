# S-02 — fpcalc startup check

## Goal
Program fails fast with a clear message if `fpcalc` is not on `$PATH`.

## Steps
1. On startup (before any processing), run `shutil.which("fpcalc")`
2. If `None` → log error + print install instructions + `sys.exit(1)`

## Install instructions to print
```
fpcalc not found. Install chromaprint:
  macOS:  brew install chromaprint
  Ubuntu: apt install libchromaprint-tools
  Windows: https://acoustid.org/chromaprint
```

## Done when
Running the CLI without `fpcalc` installed prints the above and exits 1.
Running with `fpcalc` installed proceeds normally.
