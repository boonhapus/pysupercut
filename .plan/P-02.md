# P-02 — Validate same codec across all files

## Goal
Hard error if input files have mismatched codecs — required for `-c copy` in stitch phase.

## Steps
1. After P-01, collect `{f.codec for f in video_files}`
2. If `len(set) > 1` → log each file's codec + hard error:
   ```
   Codec mismatch: re-encoding not supported. All files must share the same codec.
     a.mp4 → h264
     b.mp4 → hevc
   ```
3. Same check for container format (warn only — concat can sometimes handle mixed containers)

## Done when
Mixed-codec input exits 1 with clear message.
Same-codec input passes through silently.
