# T-01 — Containment check → DROPPED files

## Goal
Files whose entire content exists inside another file are marked DROPPED and excluded from output.

## Logic
For each file F, collect all `OverlapMatch` edges touching F.
Sum of overlapping duration in F across all matches ≥ F.duration → mark DROPPED.

More precisely: if a single match covers F's full range (range covers 0 → duration within tolerance) → DROPPED.

## Steps
1. For each file, check all its overlap ranges
2. If any single range spans `(0, F.duration)` within ±1s → DROPPED
3. Log: `"Dropping <file> — fully contained in <other_file>"`

## Done when
File B fully contained in A → B is DROPPED.
File B partially overlapping A → B is NOT dropped (handled in T-02).
