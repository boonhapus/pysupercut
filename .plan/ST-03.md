# ST-03 — Final ffmpeg concat → output file

## Goal
Concatenate all segments into the final output file using the concat demuxer.

## Command
```
ffmpeg -f concat -safe 0 -i <tempdir>/concat_list.txt -c copy <output>
```

## Steps
1. Run the above via `asyncio.create_subprocess_exec`
2. Stream stderr to structlog at DEBUG level (ffmpeg is verbose)
3. On non-zero exit → log full stderr + hard error
4. On success → print output path and file size

## Notes
- `-c copy` works here because: untrimmed segments are original codec, re-encoded temp files are both h264/aac. All inputs are the same codec going into concat.
- If `--output` path already exists → hard error before running. Do not silently overwrite.

## Done when
Output file exists after running. `ffprobe output.mp4` shows expected duration (within ±0.5s of T-05's total_output).
